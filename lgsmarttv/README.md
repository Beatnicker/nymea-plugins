# LG Smart TV

This plugin allows to connect to LG Smart TV's.

## Supported Things

* LG Smart TV
    * PIN authentication
    * Get channel, source, program
    * Set volume, mute
    * Power on/off
    * Menu navigation

## Requirements

* The device must be in the same local area network as nymea.
* The package 'nymea-plugin-lgsmarttv' must be installed.
